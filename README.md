# mdcasegen Templates

This repository contains Markdown templates for use with the mdcasegen engine.

## Structure

Templates are organized by domain and condition:

- orl/ssnhl/
- orl/conductive/

Each file represents a single case template.

## Editing Guidelines

- Do not modify metadata structure.
- Keep variation syntax intact (e.g., **option1 | option2**).
- Ensure Gold Answer section remains valid.

For full formatting rules, see TEMPLATE_GUIDELINES.md.
